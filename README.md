# Telemetry Data Processing Project

## 📌 Overview
This project processes and standardizes telemetry data from different JSON formats into a unified structure. It simulates how real-world systems handle inconsistent data inputs.

## 🚀 What I Built
- Developed functions to convert multiple data formats into a single format
- Handled timestamp conversion from ISO format to milliseconds
- Ensured consistent output structure for reliable data processing

## 🛠️ Tech Stack
- Python
- JSON

## 📂 Project Files
- main.py → Core logic for data conversion
- data-1.json → Input format 1
- data-2.json → Input format 2
- data-result.json → Expected output format

## ▶️ How to Run
Run the following command:
python main.py

## 🎯 Key Learning
This project helped me understand data transformation, handling different formats, and writing clean, structured code for real-world applications.
